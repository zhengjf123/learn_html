【照片放映 · 使用说明】

一、放图片
  1. 把你的照片文件放进本仓库的 photos 文件夹（与 manifest.json 同级）。
  2. 支持常见格式：jpg、jpeg、png、webp、gif（建议单张体积不要过大，便于手机加载）。

二、登记到放映列表
  1. 用文本编辑器打开 photos/manifest.json。
  2. 在 "photos" 数组里按顺序添加条目，每张图一条。例如：

{
  "autoAdvanceSeconds": 6,
  "photos": [
    { "file": "旅行_01.jpg", "caption": "那年夏天的风" },
    { "file": "生日合影.png", "caption": "" }
  ]
}

  · file：只写文件名，不要写 photos/ 前缀；须与 photos 文件夹里真实文件名一致（区分大小写）。
  · caption：放映时显示在照片下方的短句；不需要可以写成 "" 。
  · autoAdvanceSeconds：自动换下一张的间隔秒数，可按喜好改。

三、保存并上线
  1. 保存 manifest.json，把 photos 里的新图片一起提交并 push 到 GitHub。
  2. 等 GitHub Actions 部署完成后，刷新你的 Pages 链接；页面向下滚动即可看到「时光放映」区块。

四、页面上怎么操作
  · 自动放映：默认开启，按间隔自动切换。
  · 暂停 / 继续：点中间的播放控制按钮。
  · 上一张 / 下一张：点两侧箭头，或用键盘 ← →。
  · 胶片条：点缩略图跳到对应一张。
  · 手机：在照片上左右滑动可切换。
  · 全屏：点「全屏放映」更有影院感（浏览器会提示退出方式）。

五、若列表为空
  若 manifest 里 photos 为空，页面会显示提示，不会报错；加入照片并 push 后即显示放映。
